What is Lestrade?

Lestrade is a movement for a better web. Technically speaking it's a fully-featured, pure-JS web scaffold complete with deployment automation, package management and a robust file structure. It's used for creating efficient and scalable websites by handling all the repetitive work for you, allowing you to focus on the code rather than the trivia. It's still in development, but feel free to learn and contribute on the repository.

Features

Lestrade's streaming build system configuration is made up of multiple automated tasks. The primary ones are for templating, styles and scripts. Whilst not mentioned here, there are a collection of miscellaneous functions that help the system run such as cleaning and file copying.

Templates (pages) are written in Jade, the Node.js templating engine. After compiling the Jade templates, Lestrade parses the HTML and replaces references to non-optimized scripts or stylesheets with their compiled counterparts. The template is then tested against the W3C HTML validator and if everything's all good, it's heavily minified and a sitemap is produced.

Stylesheets in Lestrade are written in Less, a CSS pre-processor built in Node.js. Your main stylesheet is linted by Recess, Twitter's CSS quality tool. All your stylesheets are then cross-referenced with your templates and unused CSS is removed. Once this is complete, vendor prefixes are added to the compiled stylesheets. It's all minified with the auspiciously named cssmin, a CSS parser and minifier.

Scripts are written in Coffeescript, a pre-processor for Javascript that (you guessed it) runs on Node.js. Before being compiled, your Coffeescript is run through Coffeelint which helps keep your code clean and consistent. Finally, the scripts are minified and compressed with UglifyJS, a JavaScript compressor and minifier written in JavaScript.

Usage

Surprisingly, Lestrade isn't very difficult to use. All your source code goes in the "app" folder and is deployed into the "dist" folder. There are only a few functions you need to know to get started.

Download Lestrade's dependencies (you only need this once):

./primer.bat

Build your website:

gulp

Boot up the live preview server:

gulp watch

Clean up the project:

gulp clean

Bump your version number. Options are "major", "minor" and "patch":

gulp bump --type major

Download

Eventually there will be a few ways to download Lestrade, but for now let's stick with downloading from GitHub. You can either download the repo directly or clone it from GitHub with:

git clone https://github.com/haydenbleasel/lestrade.git
